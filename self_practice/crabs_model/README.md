# Leptograpsus Crabs

### Labels:
 
`SP` -- `species`

### Features:

* `sex` -- `Male` or `Female`
* `FL` -- `Frontal Lip of Carapace`
* `RW` -- `Rear with Carapace`
* `CL` -- `Length of Carapace`
* `CW` -- `Width of Carapace`
* `BD` -- `Body Length`
